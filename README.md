InitationCeremony
====

## Description
入社式に使用するPepperアプリケーション
### Features
1. 一歩前へ出る
2. 機械語自己紹介
3. 自己紹介
4. 入社への感謝
5. 入社への決意
6. 先輩方への指導依頼
7. 締めの挨拶
8. 一歩後ろへ戻る


## Install

```commandline
$ git clone git@github.com:johnokaz/InitationCeremony.git
```
